# Tic Tac Toe - Java Swing

A simple Tic Tac Toe game built using Java Swing with Object-Oriented Programming principles.

## Features

- 2 Player Mode
- Interactive GUI using Java Swing
- Winner Detection
- Draw Detection
- Score Tracking
- Restart Game Functionality
- Clean OOP Structure

## Technologies Used

- Java
- Swing
- OOP Concepts

## Project Structure

```text
src

├── Main.java
│
├── model
│   └── GameLogic.java
│
└── ui
    └── GameFrame.java
```

## How to Run

### Using IntelliJ IDEA

1. Open the project
2. Open Main.java
3. Click Run

### Using VS Code

1. Install Extension Pack for Java
2. Open the project folder
3. Open Main.java
4. Click Run Java

## Game Rules

- Player X starts first.
- Players take turns placing X and O.
- First player to get three symbols in a row wins.
- If all cells are filled and nobody wins, the match is a draw.
